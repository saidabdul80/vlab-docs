Concepts

What is Vlab

Vlab is a platform that intends to enable equitable virtual access, tools and near-real life experience of conducting basic science and engineering based experiments that have otherwise become difficult to be carried out by ALL first year or second-year science and engineering students in the laboratory within the semester period. This project is currently supported by the Royal Academy of Engineering under the Higher Education Partnerships in sub-Saharan Africa scheme (HEP-2021-129) Vlab will be standing on the shoulder of technology giants such as Laravel, Vue js and Bootstrap for development.

Installation

  1. clone the latest stable release of Vlabnigeria repository. From the command line: 
    git clone https://github.com/saidabdul80/vlab-2.git vlab
  2. Enter your newly cloned project directory using 
    cd vlab
  3. Create a file named .env file and copy the contents of the .env.example into the new .env file or alternatively rename the .env.example file to .env
  4. create database vlab
  5. edit this to yours in .env 
    DB_DATABASE= your_database_name
DB_USERNAME= your_username
DB_PASSWORD= *********
  6. From terminal/command prompt on vlab directory Run 
    composer install
    then 
    php artisan key:generate
  7. From terminal/command prompt on vlab directory Run 
    php artisan migrate:fresh --seed
  8. Run 
    yarn install && yarn watch
    or 
    npm install && npm run watch
    to install the frontend dependencies.
  9. locate 
    resources/js/plugin.js
    edit line 2 to 
    axios.defaults.baseURL = " your host address "
  10. From terminal/command prompt on vlab directory Run 
    npm run prod
  11. if you are on cpanel change root directory of your domain or subdomain to public in vlab directroy otherwise
  12. if you are on local host run 
    php artisan serve
Thank you for installing vlab
